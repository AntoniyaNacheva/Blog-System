# Blog-System

Node.js + React JS

# Getting Started Server

In the project directory:

### `cd server`

### `npm start`

# Getting Started Client

In the project directory:

### `cd client`

### `npm start`

Runs the app in the development mode.\
 Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
